# Telegram Bot Setup для SLEO

Бот отправляет напоминания волонтерам за день до события.

## Получение Telegram Bot Token

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Дайте боту имя (например: SLEO Volunteer Bot)
4. Дайте боту username (должен заканчиваться на `_bot`, например: sleo_volunteer_bot)
5. Скопируйте полученный токен (выглядит так: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)

## Конфигурация

1. Откройте `backend/.env`
2. Найдите строку `TELEGRAM_BOT_TOKEN=your-telegram-bot-token-here`
3. Замените `your-telegram-bot-token-here` на полученный токен:
   \`\`\`
   TELEGRAM_BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
   \`\`\`

## Запуск бота

\`\`\`bash
cd backend
npm install
npm run bot
\`\`\`

Бот будет:
- Запускаться каждый день в 09:00 по времени сервера
- Проверять события на завтра
- Отправлять напоминания всем волонтерам в событии

## Команды бота в Telegram

- `/start` - информация о боте
- `/events` - список твоих событий
- `/help` - справка по командам

## Важно

- Бот работает через polling (постоянное подключение)
- Убедитесь что токен правильный, иначе бот не запустится
- Убедитесь что MongoDB запущена и доступна по MONGODB_URI
- Волонтеры должны сохранить свой Telegram username в системе
